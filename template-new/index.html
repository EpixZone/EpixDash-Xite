This site is hosted on EpixNet! A decentralized, peer-to-peer web.

Getting Started

This is your site's index.html file. Everything you see here can be edited to make it your own.

  • Open the sidebar (drag from the right edge of the page) to access the file manager
  • Edit index.html to change this page
  • Upload images, CSS, and JavaScript files to build your site
  • Click Sign & Publish in the sidebar to push changes to the network

How EpixNet Sites Work

  • No servers needed your site is served directly from your browser to visitors
  • Your private key is stored locally and proves you own this site
  • content.json tracks all files and their hashes, it gets updated automatically when you sign
  • Peers who visit your site help serve it to others
As long as EpixNet is running, your site is being served to peers. The more visitors, the more available your site becomes.

Example: Adding Content

EpixNet sites are standard HTML, CSS, and JavaScript. Here's a simple example:

<!-- Add an image -->
<img src="photo.jpg" alt="My photo">

<!-- Add a link to another EpixNet site -->
<a href="/epix1readmehqfdxy4pzx7u72wwaerc4psx0gt6fety">EpixNet Docs</a>

<!-- Add a stylesheet -->
<link rel="stylesheet" href="style.css">

<!-- Add JavaScript -->
<script src="app.js"></script>

Using the EpixNet API

Your site can interact with EpixNet through the EpixFrame API. Add this to your HTML:

<script>
// Connect to the EpixNet wrapper
var page = new EpixFrame()

page.cmd("siteInfo", [], function(info) {
  console.log("Site address:", info.address)
  console.log("Peers:", info.peers)
})

// Show a notification
page.cmd("wrapperNotification",
  ["info", "Hello from my site!", 5000])
</script>
The EpixFrame API lets you read/write files, interact with the database, manage users, and more. Check the docs for the full API reference.

Tips

  • Site size limit is 10 MB by default, visitors can increase it from the sidebar
  • Use the sidebar's file manager to upload, edit, and delete files
  • After making changes, always Sign & Publish to update the network
  • You can register an xID name (like mysite.epix) for a human-readable address
